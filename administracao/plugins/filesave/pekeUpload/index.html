PekeUpload jQuery Plugin

Demos

1. - Basic configuration

Sample Code: 

$("#file").pekeUpload();

2. - Twitter bootstrap theme

Sample Code: 

$("#file").pekeUpload({theme:'bootstrap'});

3. - Restrict File Extensions

Sample Code: 

$("#file").pekeUpload({theme:'bootstrap', allowedExtensions:"jpeg|jpg|png|gif"});

4. - Single file upload

Sample Code: 

$("#file").pekeUpload({theme:'bootstrap', multi: false});

5. - Alert on error

Sample Code: 

$("#file5").pekeUpload({theme:'bootstrap', allowedExtensions:"jpeg|jpg|png|gif", onFileError:function(file,error){alert("error on file: "+file.name+" error: "+error+"")}});
