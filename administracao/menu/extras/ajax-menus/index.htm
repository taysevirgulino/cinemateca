Here is our Ajax Based Menus

They are incredibly easy to use, all you need to do is declare a <script> Reference to the file mm_browserapi.js inside your HTML, this should go just after the reference to mmenudom.js.

Then, you call your ajax files using a menu item similar to this: aI("showmenu=ajax:my-milonic-menu.txt;text=My Milonic;");<br>
The contents of the file samples-menu.txt is as follows, but could be dynamically written using any server side scripting language such as PHP, ASP, Cold Fusion or JSP etc etc:

with(milonic=new menuname("MyMilonic")){
style=menuStyle;
aI("text=Login;url=http://www.milonic.com/login.php;");
aI("text=Licenses;url=http://www.milonic.com/mylicenses.php;");
aI("text=Invoices;url=http://www.milonic.com/myinvoices.php;");
aI("text=Make Support Request;url=http://www.milonic.com/reqsupport.php;");
aI("text=View Support Requests;url=http://www.milonic.com/mysupport.php;");
aI("text=Your Details;url=http://www.milonic.com/mydetails.php;");
}
